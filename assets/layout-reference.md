# 布局参考：Section 模板与间距指南

## 网格系统

```css
/* 基础容器 */
.container {
  width: 100%;
  max-width: 1100px;
  margin: 0 auto;
  padding: 0 24px;
}

/* 两栏：内容 + 侧边 */
.layout-2col {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 64px;
  align-items: center;
}

/* 三栏：项目/技能卡片 */
.layout-3col {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 24px;
}

/* 非对称精选布局 */
.layout-featured {
  display: grid;
  grid-template-columns: 2fr 1fr;
  gap: 24px;
}

@media (max-width: 768px) {
  .layout-2col,
  .layout-3col,
  .layout-featured {
    grid-template-columns: 1fr;
    gap: 24px;
  }
}
```

**间距：**
- 顶部内边距：`120px`（导航高度 + 缓冲）

- Tagline 到 CTA：`48px`
- 最小高度：`100vh`

## 间距单位参考

| Token | 数值 | 使用场景 |
|---|---|---|
| `--space-1` | `4px` | 图标间距、紧凑标签 |
| `--space-2` | `8px` | 相关元素之间 |
| `--space-3` | `12px` | 按钮垂直内边距 |
| `--space-4` | `16px` | 卡片间距、列表间距 |
| `--space-6` | `24px` | 按钮水平内边距、列间距 |
| `--space-8` | `32px` | 卡片内边距 |
| `--space-12` | `48px` | Section 内部块间距 |
| `--space-16` | `64px` | 主要区块间距 |
| `--space-24` | `96px` | Section 垂直内边距（最小值） |
| `--space-32` | `128px` | 全出血 Section 内边距 |

## 响应式断点

```css
/* 移动端优先 */
/* 默认：375px+ */
@media (min-width: 640px)  { /* 竖屏平板 */ }
@media (min-width: 768px)  { /* 横屏平板 */ }
@media (min-width: 1024px) { /* 桌面 */ }
@media (min-width: 1280px) { /* 宽屏桌面 */ }
```

**关键响应式变化：**
- 导航：移动端折叠（只保留 Logo + 单个 CTA，或汉堡菜单）
- Hero 姓名：`clamp(2.5rem, 10vw, 5rem)` 流畅缩放
- 三栏网格 → 移动端单栏
- Section 内边距：桌面 `96px 24px` → 移动端 `64px 20px`
- 关于我：移动端堆叠（头像在文字上方）
